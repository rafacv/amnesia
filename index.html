amnesia by rafaelcv

Amnesia eases web development by reloading WSGI apps and imported modules before each request.

Amnesia reloads modules on-the-fly as an aid for development of WSGI compliant applications.

Amnesia is meant to be used like a standalone executable, running on the shell like:

$ amnesia mymodule my_wsgi_app [-p portno] [-s hostname]

or as a WSGI Middleware as follows:

>>> from wsgiref.simple_server import make_server
>>> from amnesia import Amnesia
>>> 
>>> my_reloading_app = Amnesia("mymodule", "wsgi_app")
>>> 
>>> if __name__ == "__main__":
>>>     httpd = make_server("", 8080, my_reloading_app)
>>>     print("Running on localhost:8080...")
>>>     print("Modify your app and refresh your browser's page.")
>>>     httpd.serve_forever()
>>>

Install

$ easy_install amnesia

or

$ curl http://github.com/rafaelcv/amnesia/tarball/v0.2.0 -Lo amnesia.tar.gz
$ tar xzf amnesia.tar.gz
$ cd amnesia
$ python setup.py install
